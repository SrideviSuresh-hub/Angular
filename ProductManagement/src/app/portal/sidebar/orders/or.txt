To achieve the desired styling for your `ConfirmDialog`, you can use custom CSS. Here's how you can apply the styles:

1. **Center the error icon at the top.**
2. **Center-align the dialog message.**
3. **Style the cancel button with a gray background and the delete button with a blue background.**

Here's the updated HTML and CSS:

### HTML/* /* 

```html
<p-confirmDialog key="deleteConfirm">
  <ng-template pTemplate="header">
    <div class="dialog-header">
    </div>
  </ng-template>

  <ng-template pTemplate="message">
    <div class="dialog-message">div class col-12

      <i class="pi pi-times-circle error-icon"></i>
      <h3 class="dialog-title">Are You sure!??</h3>
      <p>Do you really want to delete the selected item?</p>
    </div>
  </ng-template>

  <ng-template pTemplate="footer">
    <div class="dialog-footer">
      <p-button type="button" pButton label="Cancel" icon="pi pi-times" class="custom-cancel-btn" (click)="onReject()"></p-button>
      <p-button type="button" pButton label="Delete" icon="pi pi-check" class="custom-delete-btn" (click)="onAccept()"></p-button>
    </div>
  </ng-template>
</p-confirmDialog>
```

### CSS

```css
.dialog-header {
  text-align: center;
}

.error-icon {
  font-size: 2rem;
  color: red;
}

.dialog-message {
  text-align: center;
}

.dialog-footer {
  display: flex;
  justify-content: space-between;
}

.custom-cancel-btn {
  background-color: gray;
  border: none;
  color: white;
}

.custom-delete-btn {
  background-color: blue;
  border: none;
  color: white;
}
```
